#### Access Specifiers

*   In C++, there are 3 access specifiers:

*   **Private inheritance:**

        | Accessibility | Private Members | Protected Members | Public Members |
        |---------------|-----------------|-------------------|----------------|
        | Base Class    | abcd            | abcd              |                |
        | Derived Class | abcd            | abce              |                |

*   **Protected inheritance   :**

        | Accessibility | Private Members | Protected Members | Public Members |
        |---------------|-----------------|-------------------|----------------|
        | Base Class    | abcd            | abcd              |                |
        | Derived Class | abcd            | abce              |                |

*   **Public inheritance:**

        | Accessibility | Private Members | Protected Members | Public Members |
        |---------------|-----------------|-------------------|----------------|
        | Base Class    | abcd            | abcd              |                |
        | Derived Class | abcd            | abce              |                |
